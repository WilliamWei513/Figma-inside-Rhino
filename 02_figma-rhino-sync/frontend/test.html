📋 安装步骤：

1. 确保服务器正在运行：
cd 02_figma-rhino-sync/backend/local-server
npm start

2. 打开 Figma Desktop 应用
3. 进入 Plugins → Development → Import plugin from manifest...
4. 选择 02_figma-rhino-sync/frontend/manifest.json
5. 运行插件：Plugins → Development → Rhino to Figma Sync
